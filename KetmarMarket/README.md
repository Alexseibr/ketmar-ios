# KetmarMarket iOS App

Создайте Xcode проект здесь.

## Рекомендуемая структура:

```
KetmarMarket/
├── KetmarMarket.xcodeproj
├── KetmarMarket/
│   ├── App/
│   │   └── KetmarMarketApp.swift
│   ├── Views/
│   ├── Models/
│   ├── Services/
│   │   └── APIClient.swift
│   ├── ViewModels/
│   └── Resources/
│       └── Assets.xcassets
└── KetmarMarketTests/
```

## Создание проекта:

1. File → New → Project
2. iOS → App
3. Product Name: KetmarMarket
4. Interface: SwiftUI
5. Language: Swift
6. Bundle ID: by.ketmar.market
